📊 Project Overview

This project is a comprehensive Business Intelligence (BI) solution developed using Microsoft Power BI to transform raw sales data into meaningful business insights. The dashboard provides an end-to-end analysis of sales performance, profitability, inventory, customer behavior, and regional trends through interactive visualizations and dynamic reporting.

The project covers the complete analytics workflow, including data cleaning with Power Query, data modeling, DAX measure creation, KPI development, and interactive dashboard design. It enables users to monitor business performance, evaluate key metrics, identify top-performing products and regions, analyze yearly trends, and explore data from multiple perspectives using advanced filtering and drill-down capabilities.

The solution consists of three interactive dashboards:

Sales Dashboard – Provides an overview of sales performance, quantities, yearly trends, top-selling products, and regional sales distribution.
Profit Dashboard – Focuses on profitability analysis by comparing sales, costs, and profits across products, regions, and time periods.
Details Dashboard – Delivers detailed operational insights through comprehensive tables, geographic analysis, buying group distribution, tax analysis, and performance metrics.

This project demonstrates practical Business Intelligence skills by combining Power BI, Power Query, DAX, data modeling, KPI design, and data visualization to support data-driven decision-making and business performance optimization.
